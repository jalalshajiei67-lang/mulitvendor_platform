## Step 2: Test Connection (30 seconds)

```bash

./test-connection.sh
```

## Step 3: Deploy (2 minutes)

### On Windows:
```bash
deploy-windows.bat
```

### On Linux/Mac:
```bash
chmod +x deploy.sh
./deploy.sh
```

## Step 4: Start Services on VPS (1-2 minutes)


```bash
docker-compose exec backend python manage.py createsuperuser
```
